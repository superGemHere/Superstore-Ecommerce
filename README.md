## Installation Client

1. Clone the repository:
   ```bash
   git clone https://github.com/superGemHere/Superstore-Ecommerce.git
   ```
2. Navigate to the client directory:
   ```bash
   cd client
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
